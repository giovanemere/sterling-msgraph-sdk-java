# Guía de Desarrollo - Sterling Msgraph Sdk Java

## 🛠️ Configuración del Entorno

### Prerrequisitos
- Java 11+
- Maven o Gradle
- Git

### Instalación
```bash
git clone https://github.com/giovanemere/sterling-msgraph-sdk-java.git
cd sterling-msgraph-sdk-java
./mvnw clean install
```

## 🧪 Testing

```bash
# Ejecutar tests
./mvnw test
```
